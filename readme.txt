КАК ОТКРЫТЬ:
  1. Разархивируйте ZIP в любую папку.
  2. Откройте index.html в браузере (Chrome, Firefox, Edge).
  3. Нажмите плитку «Игровая комната» — откроется games.html.
  Все 7 файлов должны быть в ОДНОЙ папке.
